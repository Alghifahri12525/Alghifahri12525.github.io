Tentang kami

Ini merupakan website pribadi saya yang pertama kali saya buat, saya gak berharap banyak tentang keahlian saya, tetapi saya berharap bisa membangun sebuah website yang lebih bagus dari ini dan bisa digunakan untuk semua orang. Selain membuat website, saya juga berharap membangun aplikasi lainnya, namun itu masih terlalu jauh dan masih banyak yang saya harus pelajari. Saya saat ini hanya bisa menggunakan beberapa bahasa pemrograman seperti JavaScript, Html, dan CSS. Dan untuk frameworknya saya menggunakan bootstrap.

Saya juga merupakan seorang anak smp yang baru tamat sekolah. Sekarang saya sedang melanjutkan jenjang yang lebih lanjut yakni sma, saya hanya berharap masa SMA saya baik-baik saja dan dapat memberi manfaat kepada siapa saja. Untuk website ini saya hanya membuat untuk pengenalan diri saja, dan di sautu saat nanti website ini akan saya ubah yang dari awalnya melihat video tutorial dan kedepannya saya akan membuat atas ide saya sendiri.
